# Florence Student — 网页项目完整规格说明

> **项目类型**: 响应式 Web 应用（H5 / PWA）
> **视觉风格**: 手绘叙事风格（Hand-Drawn Narrative）—— 本项目唯一的设计方向
> **目标用户**: 佛罗伦伦萨大学（UNIFI）学生
> **核心功能**: 课程搜索与评价、教授信息浏览、学习资源共享、选课决策辅助
> **界面语言**: 意大利语（正文）+ 英文（课程编码）
> **数据支持**: Supabase 后端（BaaS）+ 匿名认证

---

## 一、项目概述

Florence Student 是一个面向佛罗伦萨大学（UNIFI）学生的课程评价与资源共享平台。学生可以搜索浏览课程、查看评分看板、发布匿名评价、下载学习资料，在真实同伴经验中做出更明智的选课决策。

### 核心价值

- **学术发现**: 按专业分组浏览课程，支持搜索和筛选
- **同伴智慧**: 三维评分体系（难度 / 给分质量 / 讲课质量）+ 考试技巧分享
- **资源共享**: 分类资料库（讲义 / 练习 / 历年真题 / 音频）+ 社交证明
- **个人中心**: 评价历史、收藏课程、个人资料管理

### 设计理念

整个应用像一本**手写笔记**——每一门课程是一张被认真记录的黄铜钉笔记页，每一次评价都是便签纸上的真诚留言。温暖、真诚、有人情味，而不是冰冷的数据库界面。

### 技术架构

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 前端 | HTML5 + CSS3 + Vanilla JS | 无框架依赖，便于 AI 生成 |
| 后端 | Supabase | PostgreSQL + Auth + Storage |
| 数据 | REST API（Supabase） | 匿名用户可读，登录后可写 |
| 部署 | 静态托管 | Vercel / Netlify / Cloudflare Pages |

## 二、视觉设计体系

### 2.1 核心理念

**像一本手写的笔记本**——不是设计完美的数字产品，而是带有温度的纸笔记录感。

核心原则:
- 有机的不完美感：微倾斜、轻微旋转、手写质感
- 纸张质感：暖白底色、纸张纹理、胶带贴纸
- 真实感：便签纸、图钉、便笺、手写字体
- 对比色点缀：修正液红 + 荧光蓝绿 + 深铅笔黑

### 2.2 颜色系统

```css
:root {
  /* === 全局背景与表面 === */

  --surface-low:     #f7f3f2;   /* 低层级容器 */

  --surface-high:    #e5e2e1;   /* 高层级容器 */

  /* 用于标题、正文、描边——像墨水渗入纸张 */

  /* CTA按钮、重要交互、选中态 */
```

### 2.3 字体


### 2.5 阴影

手绘风格的阴影是**硬投影**——不像现代扁平设计的柔和阴影，更像是纸张层叠产生的真实投影。

```css
/* 卡片阴影——3px 硬偏移，像纸片浮在背景上 */
.card {
  box-shadow: 3px 3px 0 rgba(24,25,25,0.12);
}

/* 大卡片双层阴影 */
.card-elevated {
  box-shadow: 3px 3px 0 rgba(24,25,25,0.15);
  box-shadow: 0 4px 12px rgba(24,25,25,0.06);
}

/* 按钮阴影——更重的偏移 */
.btn-primary {
  box-shadow: 4px 4px 0 rgba(24,25,25,0.35);
}

/* 关闭阴影（按下状态） */
.btn-primary:active {
  box-shadow: 1px 1px 0 rgba(24,25,25,0.25);
  transform: translate(2px, 2px);
}
```

### 2.6 手绘装饰元素

#### 胶带（Tape）

用在卡片角落的装饰性胶带，模拟真实笔记页的视觉效果。

```css
.tape {
  position: absolute;
  top: -10px;
  width: 80px;
  height: 20px;
  background: var(--tape);
  border-radius: 2px;
  transform: rotate(-3deg);  /* 微倾斜 */
  opacity: 0.85;
  box-shadow: 1px 1px 2px rgba(0,0,0,0.1);
}
```

**使用场景**: 快捷统计卡片（Media 评分卡片）、特色卡片

#### 旋转装饰

微角度旋转让卡片看起来像随手放在桌上的笔记页。

```css
/* 不同卡片使用不同的小角度旋转 */
.card:nth-child(1) { transform: rotate(-1deg); }
.card:nth-child(2) { transform: rotate(2deg); }
.card:nth-child(3) { transform: rotate(-2deg); }
```

#### 便签纸效果

用较深的纸张色作为背景，模拟便签纸贴在页面上的效果。

```css
.sticky-note {
  background: #fdf9e6;  /* 便签纸黄 */
  border-left: 4px solid var(--warning);  /* 荧光边 */
  padding: 16px;
  transform: rotate(-0.5deg);  /* 轻微倾斜 */
  box-shadow: 2px 3px 8px rgba(0,0,0,0.1);
}
```

**使用场景**: 考试提示区（Consigli per l'esame）、空状态说明

#### 图钉装饰

用圆形 + 阴影模拟图钉钉住笔记页的效果。

```css
.thumbtack {
  width: 16px;
  height: 16px;
  background: radial-gradient(circle at 40% 40%, #ff6b6b, #c0392b);
  border-radius: 50%;
  box-shadow: 1px 2px 3px rgba(0,0,0,0.3);
  position: relative;
}
.thumbtack::after {
  content: '';
  position: absolute;
  bottom: -4px;
  left: 50%;
  transform: translateX(-50%);
  width: 2px;
  height: 6px;
  background: #888;
}
```

**使用场景**: 专业区块头部、分类标题

## 六、数据模型

### 6.1 数据库表（Supabase PostgreSQL）

```sql
-- profiles（用户资料）
{
  id: UUID (PK, 关联 auth.users),
  username: TEXT,
  faculty: TEXT,
  enrollment_year: INTEGER,
  avatar_url: TEXT,
  created_at: TIMESTAMPTZ
}

-- courses（课程）
{
  id: UUID (PK),
  name: TEXT,
  faculty: TEXT,
  credits: INTEGER,
  description: TEXT,
  semester: TEXT,
  is_required: BOOLEAN,
  year_level: INTEGER,
  program_code: TEXT,
  professor_id: UUID (FK → professors),
  created_at: TIMESTAMPTZ
}

-- professors（教授）
{
  id: UUID (PK),
  name: TEXT,
  department: TEXT,
  faculty: TEXT,
  bio: TEXT,
  created_at: TIMESTAMPTZ
}

-- reviews（评价）
{
  id: UUID (PK),
  user_id: UUID (FK → profiles),
  course_id: UUID (FK → courses),
  professor_id: UUID (FK → professors, nullable),
  username: TEXT,
  score_difficulty: INTEGER (1-5),
  score_grading: INTEGER (1-5),
  score_teaching: INTEGER (1-5),
  content: TEXT,
  exam_tips: TEXT,
  is_anonymous: BOOLEAN,
  helpful_count: INTEGER,
  created_at: TIMESTAMPTZ
}

-- resources（资源）
{
  id: UUID (PK),
  course_id: UUID (FK → courses),
  uploader_id: UUID (FK → profiles),
  title: TEXT,
  type: TEXT (dispense / esercizi / esami / appunti / audio),
  file_url: TEXT,

  download_count: INTEGER,
  created_at: TIMESTAMPTZ
}

-- user_favorites（收藏）
{

  course_id: UUID (FK → courses),
  created_at: TIMESTAMPTZ
}

{
  code: TEXT (PK, 如 "LM-92"),

  faculty: TEXT,
  description: TEXT,
```
